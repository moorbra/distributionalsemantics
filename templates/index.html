Welcome!

This site is a prototype for an enterprise level application for text analytics.

The concept:

  • The system could be deployed in an enterprise or offered as software as a service (SaaS).
  • A system administrator can maintain the model through the application.
  • The complexities of semantic modelling are abstracted from the administrator.
  • The application could have many corpus providers. It is currently integrated with Wikipedia.
  • The application can create a model from a corpus.
  • Models can get built for specific domains (ex: mortgage industry, automobile repair, cloud management).
  • Models can get distributed and imported into the system.
  • A model can get integrated into a knowledge management system (KMS).
  • A semantic model can improve a KMS by auto-indexing, improving retrieval with semantic similarity queries, integrating into data driven workflow.

Technology Stack
Technology Role Reference
Python Programming language widely used for natural language processing tasks. https://www.python.org
